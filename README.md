## 这是一个Angular注入依赖的练习

  1.注入依赖的流程
    （1）通过import根据文件路径导入服务
    （2）在Module的providers中配置注入或者通过providedIn配置注入器
    （3）在构造函数中注入依赖

  2.用命令创建服务： ng g service [name]
  
  3.先编写服务中的工具类函数

  4.将组件中的函数完善
